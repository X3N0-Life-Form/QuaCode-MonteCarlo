QuaCode-MonteCarlo
==================

Compilation:
------------
Requirements:
  - cmake
  - [boost](http://www.boost.org/) (tested with boost 1.56).

Instructions:
  - In the project root, call `cmake`
  - If the Makefile creation was successful, call `make`
  - The generated binary should be called `QC-MC`
